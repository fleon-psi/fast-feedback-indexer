## Simple test files

Installed by turning on the *INSTALL_SIMPLE_DATA_FILES* option for cmake.

### Structure

```
| Syntax element                        | Comment                                                                  |
|:--------------------------------------|:-------------------------------------------------------------------------|
| input = unit_cell spot_vector         | numbers are white space/line separated, comment lines can be anywhere    |
|---------------------------------------+--------------------------------------------------------------------------|
| unit_cell = vec vec vec               | real space unit cell on the first line of the input file                 |
|---------------------------------------+--------------------------------------------------------------------------|
| vec = x y z                           |                                                                          |
|---------------------------------------+--------------------------------------------------------------------------|
| spot_vector = vec+                    | on line per spot in reciprocal space                                     |
|---------------------------------------+--------------------------------------------------------------------------|
| comment_line = # comment              |                                                                          |
```

### Data

```
| File                                  | Comment                                                                  |
|:--------------------------------------|:-------------------------------------------------------------------------|
| test0.txt - test2.txt                 | Purely artificial and perfect sets (i.e. all points are produces by      |
|                                       | multiplying expected lattice vectors, vectors in the first line are also |
|                                       | the actual solution)                                                     |
|---------------------------------------+--------------------------------------------------------------------------|
| image*.txt                            | Files generated by three spotfinding algorithms on actual protein images |
|                                       | (i.e. they might contain peaks that are artifacts and not part of an     |
|                                       | actual lattice), the actual vectors (gold answer) should be              |
|                                       | approximately:                                                           |
|                                       | a =  39.431335      25.273994      63.585350                             |
|                                       | b =  28.513729      60.642746     -41.786659                             |
|                                       | c = -29.096014      20.499205       9.895323                             |
```
